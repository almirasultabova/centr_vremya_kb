# Монетизация моих инициатив

> Этот файл — для фиксации того, что ты создаёшь и какую ценность это несёт центру.
> Используй для переговоров о зарплате, бонусах или проценте.

## Зачем это нужно

Ты делаешь работу нескольких людей. Если не фиксировать — это невидимо.
Фиксируй → оцениваешь → обосновываешь ценность → разговариваешь о деньгах с аргументами.
