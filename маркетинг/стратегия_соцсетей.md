## ⚡ Смена стратегии с июня 2026 (решение Наташи)

**Старый подход:** много продающих постов с ценами.
**Новый подход:** value-first — польза, охват, вирусный контент. Продающие — реже и тоньше.

### Пропорция постов
- **70% — польза / охват / вирусный** (форумные боли, разборы мифов, инсайты, истории, «продажа эмоций»)
- **20% — социальное доказательство** (отзывы, кейсы, истории клиентов от лица центра) — **2 поста в неделю минимум**
- **10% — продающие** (мероприятия, диагностика, курсы) — только в формате «история → продукт в конце»

### Принципы
- **Продаём эмоции, не услуги.** «Снова спать спокойно» вместо «10 сессий по методу X». Цена и название курса — мелким шрифтом в конце.
- **Цена в обычных постах не пишется.** Без давления.
- **AI-озвучка отзывов запрещена.** Только текст-на-видео или живые селфи-видео клиентов.
- **3–4 рилза в неделю** минимум, чтобы запустить алгоритм Instagram (по методу Андреева, см. [`методология_контента_ai.md`](методология_контента_ai.md)).

### Лид-магниты
- **Диагностика за 500₽** — низкий порог, прямой контакт со специалистом. Реклама готовится отдельно.
- **1-й онлайн-поток «Счастья» 06.06** со скидкой 20% — главный продающий блок июня.

## Instagram — витрина доверия

### Цель
Создать ощущение: «Это место, где мне безопасно. Здесь я могу быть собой»

- **Эфиры** — редкие, но тёплые

### Рубрики

| Рубрика | Суть |
|---------|------|
| «Психология для жизни» | Мини-инсайты, которые человек узнаёт в себе |

## Тональность (Tone of Voice) в соцсетях

**Instagram:** Мягкий. Человеческий. Осознанный. Без давления. Без инфоцыганства. Как разговор с человеком, который рядом.

**Telegram:** Ещё теплее, чем Instagram. Много «мы рядом».

**Оба канала:**
- Обращение: «вы» (с маленькой буквы)
- Тон: тёплый, уверенный, человечный — не холодный, не канцелярский
- Стиль: живой, простой, без пафоса и эзотерики
- Позиция: «Это не клиника. Это живое пространство»
- Слоган: «Раскрой свою бесконечность» / символ ∞

## Принципы контент-планирования

### Пятничный обзор выходных
Каждую пятницу — один пост, который охватывает ВСЕ события субботы и воскресенья. Связан нарративно (общая тема), а не перечислением. Нет отдельных «Сегодня» для выходных — пятничный обзор и есть последнее напоминание.

### Перекрёстные упоминания похожих событий
Если в месяце есть тематически близкие мероприятия — они упоминают друг друга. **Не рекламным блоком**, а одной фразой в хвосте поста. Принцип: главное CTA — основное событие, упоминание второго — мягкое, без давления.

**Как определить, что события «похожи»:**
- Одна целевая аудитория (например, все мероприятия для специалистов МАК)
- Один тип боли (тело → Даосские пульсации и телесные запросы Время для женщин)
- Один уровень запроса (поверхностный → Т-игра, глубокий → расстановки)
- Прямая связь «вход → следующий шаг» (обучение МАК → супервизия МАК)

**Типовые кластеры центра «Время»:**

| Кластер | События | Как связывать |
|---------|---------|--------------|
| МАК-инструменты | Супервизия МАК + Обучение МАК | «Хотите освоить с нуля — обучение. Уже работаете — супервизия» |
| Женский/телесный | Даосские пульсации + Время для женщин | «Тело + разговор — разные форматы, одна тема» |
| Родители + подростки | Время для родителей + Курс подростков | «Пока ребёнок на курсе — есть встреча и для вас» |
| Паттерны/системы | Т-игры + Расстановки | «Игра — вспышка. Расстановки — глубина» |

**Формула фразы-перекрёста (в хвосте поста):**
> «А если хотите [другой угол той же темы] — у нас [событие, дата, цена]. [Мягкий призыв]»

**Примеры готовых фраз:**
- «А если давно хотели освоить МАК с нуля — 16 апреля старт обучения. Расскажем подробнее»
- «Уже работаете с МАК? Тогда — супервизия. Следующая встреча 22.04, пишите в директ»
- «Игра даёт вспышку осознания. Если хотите разобраться, откуда это идёт — в субботу расстановки»
- «А завтра — Время для родителей онлайн. Специально о стрессе в семье. Всего 500 ₽»

### Нарративная цепочка недели
У каждой недели — своя тема. Посты внутри недели развивают историю, а не стоят отдельными объявлениями. Подписчик должен чувствовать: «Они рассказывают про одно и то же, только с разных сторон».

### Маркетинговые фреймворки
- **Лестница Ханта (6 ступеней)** — каждый пост двигает аудиторию на одну ступень вперёд
- **5-дневный прогрев** — для продуктов от 4 500 ₽: специалист → мифы → продукт → кейс → возражения
- **Путь героя** — для постов знакомства со специалистами
- **Метод 4U** — для первой строки каждого поста (проверяй: Uniqueness, Usefulness, Ultra-specificity, Urgency)
- **Триггеры Шугермана + Чалдини** — выбирать триггер под ступень Ханта и тип поста
