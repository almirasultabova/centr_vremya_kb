# Групповые программы (матрица «Время для...»)

## Время для подростков

- **Аудитория:** 10–17 лет (две группы: 10–13 и 13–17)
- **Формат:** очно, курс 4 месяца (4 модуля по 4 занятия)
- **Цена:** 4 500 ₽/мес
- **Темы:** самопознание, эмоции, границы, стресс, отношения
- **После курса:** поддерживающие встречи раз в 2 недели
- **Ведущие:** Анастасия Полыгалова, Екатерина Мовчан, Олеся Зайчикова, Галина Кучеренко
- **Примечание:** самое стабильное и прибыльное направление

## Время для женщин

- **Формат 1:** еженедельные встречи — 2 000 ₽ разово / 1 500 ₽ по абонементу
- **Формат 2:** онлайн-подписка — 990 ₽/мес (эфиры, практики, чат, материалы)
- **Темы:** уверенность, принятие, женственность, отношения
- **Ведущие:** Татьяна Даниленко, Екатерина Мовчан, Олеся Зайчикова

## Время для счастья

- **Формат:** группа, 3 месяца, онлайн
- **Суть:** личностный рост

## Время для стройности

- **Формат:** групповой коучинг

## Время для психосоматики

- **Формат:** онлайн, 1 месяц

## Время для родителей

- Групповой формат

## Время для расстановок

- Системная работа с семьёй, родом, отношениями

## Время для игр (МАК)

- Трансформационные игры, МАК-встречи, девичники

## Время для бизнеса (B2B)

- Корпоративные тренинги, бизнес-расстановки
- Отдельный цикл продаж

## Обучение

- **Обучение МАК-картам:** 15 000 ₽
- **Телесные практики и чаши:** 3 500 ₽

## Воронка клиента

```
Вход (игры, зарядка, эфиры)
  ↓
Основной продукт (консультация / группа)
  ↓
Комьюнити / подписка
  ↓
Обучение / B2B
```
