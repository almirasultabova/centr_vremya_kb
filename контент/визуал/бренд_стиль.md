## Типографика

- Заголовок: крупный (80–84px в 1080px карточке), bold. Первая строка — тёмная, вторая — фиолетовая
- Подзаголовок: 21–22px, обычный, цвет MID
- Пункты: 20–21px bold (заголовок) + 19–20px обычный (подпись)
- CTA: 28–30px bold белый (контакт) + 17–18px светло-фиолетовый (подсказка)

## Структура карточки 1080×1080

```
┌─────────────────────────────────┐
│ [Логотип PNG]          y=48     │
│                                 │
│ НОВАЯ УСЛУГА (eyebrow)  y=158   │
│ ▬▬▬ (акцент-линия)     y=184   │
│                                 │
│ Заголовок тёмный        y=204   │
│ Заголовок фиолетовый    y=300   │
│                                 │
│ Подзаголовок строка 1   y=414   │
│ Подзаголовок строка 2   y=448   │
│                                 │
│ • Пункт 1               y=516   │
│ • Пункт 2               y=612   │
│ • Пункт 3               y=708   │
│                                 │
│ [Плашка цена]           y=822   │
│                                 │
├─────────────────────────────────┤
│ CTA: @VREMYA_DMD        y=952   │  ← тёмно-фиолетовый баннер
└─────────────────────────────────┘
```

**Что НЕ делать:**
- Горизонтальные линии-разделители через всю карточку (выглядят как ненужная черта)
- Вертикальные декоративные полосы по краю
- Рисованные «листья» через PIL — выглядят как шашлык
- Слишком мелкий текст в CTA (минимум 17px)

**Декор — только:**
- Полупрозрачные круги (радиальный градиент, opacity 8–18%)
- Знак ∞ как watermark справа (opacity ~8%)
