## 1. Цели сайта

Сайт решает четыре задачи одновременно:

1. **Запись клиентов** — основной поток через сайт ведёт на диагностику 500₽, индивидуальную консультацию или групповую программу. KPI: количество записей через сайт в месяц.
2. **Представительство и доверие** — позиционирование как центра уровня премии Губернатора, 11 лет работы, 4000+ клиентов. Не «частный психолог».
3. **SEO-витрина** — органический трафик из Яндекса по запросам «психолог Домодедово», «детский психолог Москва», «семейный психолог онлайн» и т.п. Для этого предусмотрены статьи, страницы направлений, отдельные страницы специалистов.
4. **Каталог специалистов** — посетитель может самостоятельно выбрать психолога через фильтр (по аналогии с интернет-магазином). Это решает основной барьер: «у центра 13 специалистов, к кому идти?».

Объём работ значителен, поэтому проект разбит на этапы (см. раздел 6).

## 6. Этапы запуска

Проект разбит на этапы по приоритету бизнес-ценности:

### Этап 0 — фундамент (1–2 недели)
- Согласование структуры, дизайна, прототипа логотипа
- Закупка домена / хостинга
- Выбор стека и CMS
- Решение по мультифилиальной архитектуре

### Этап 1 — MVP, главное (3–4 недели)
- Главная страница (hero, направления, специалисты превью, отзывы, форма)
- Каталог специалистов с фильтром (минимальный набор фильтров)
- Страница каждого специалиста (≈13 страниц)
- Страница «Индивидуальная консультация» с ценами и CTA на диагностику 500₽
- Афиша (перенос с улучшениями)
- Контакты + карта
- Базовый блог-движок (без статей пока)
- Форма обратной связи + интеграция с Telegram-уведомлением
- Яндекс.Метрика + цели

### Этап 2 — расширение (4–6 недель)
- Страницы остальных направлений (детский, подростковый, семейный, клинический, перинатальный, расстановки, для бизнеса)
- Страница «Цены» (общий прайс)
- Страница «О центре» (история, награды, гос. лицензия)
- Страница «Академия психологии» (B2B направление)
- Раздел «Отзывы» — полная страница со всеми отзывами
- Первые 5–10 статей в блоге
- Виджет онлайн-записи (YClients)

### Этап 3 — мультифилиальность и оптимизация (по мере открытия филиалов)
- Selector филиала вверху
- Адаптация каталога специалистов под филиал
- SEO-оптимизация под локальные запросы
- A/B-тесты главных CTA
- Регулярные публикации в блог (≥2 статьи в месяц)

## 7. Что предоставляет заказчик

- Все тексты сайта (готовятся силами заказчика на основе базы знаний)
- Все фото специалистов и центра (требуется фотосессия — отдельная задача)
- Видео-материалы (при наличии — предоставляются заказчиком)
- Список отзывов в текстовом виде (есть в [отзывы/отзывы.md](../отзывы/отзывы.md))
- Брендбук: цвета, шрифты, логотип в векторе (статус на 2026-05-14 — есть только PNG)
- Согласия на обработку персональных данных (юридические тексты)

## 8. Что не входит в первую версию

Границы первой версии:

- Личный кабинет клиента
- Онлайн-оплата на сайте (платежи только через YClients или прямой перевод)
- Многоязычность (только русский)
- Тёмная тема
- Чат с психологом онлайн (только мессенджеры через ссылки)
- Маркетплейс с другими центрами
