# Расписание центра психологии ВРЕМЯ

## Источник

Google Таблицы (актуальное расписание):
https://docs.google.com/spreadsheets/d/1f9-NbK6FSsCstC6AQ2c7NESIO07RBMyeyl1shNZragw/edit

**Как читать:** таблица многолистовая — каждый лист = месяц.
**Доступ:** требует авторизацию Google. Прямой экспорт CSV работает только для первого листа.
**Что делать:** просить Альмиру скинуть актуальное расписание текстом или скриншотом, если нужны данные за текущий месяц.

## Структура таблицы

| Колонка | Что содержит |
|---------|-------------|
| Дата | ДД.ММ |
| День | Пн/Вт/... |
| Время | ЧЧ:ММ |
| Формат | ОЧНО / ОНЛАЙН |
| Статус | Прошло / Запланировано / В процессе |
| Контент-план | Опубликовано / Не готов |
| Комментарий | Доп. пометки |
| Название | Название события |
| Тема | Тема встречи |
| Ведущий | Специалист |
| Стоимость | Цена |
| Результат | Итог (сколько пришло, отменено и т.д.) |
